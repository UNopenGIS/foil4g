# openmaptiles/openmaptiles-tools

OpenMapTiles のためのツール群です。[[openmaptiles/openmaptiles]] や同様のプロジェクトで定義されたレイヤーに基づいて、Imposm マッピングや SQL ファイルを準備するツールが含まれています。また、マップデータのダウンロード、解析、デバッグ、パフォーマンス評価ツールも含まれています。
