# VIRGO ビエンチャン 道路決定データ

## データソース情報

| 項目             | 内容                                                               |
| ---------------- | ------------------------------------------------------------------ |
| **データID**     | determiningroad_v0                                                 |
| **ライセンス**   | 不明                                                               |
| **データ形式**   | Shapefile                                                          |
| **ファイル形式** | SHP                                                                |
| **提供元**       | Vientiane Integrated Urban Information GIS-based Opendata Platform |
